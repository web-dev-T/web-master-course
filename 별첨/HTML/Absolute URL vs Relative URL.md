##### Absolute URL 표현하기

- `/`로 시작하는 경우 (동일 서버의 루트 레벨에 대하여 파일을 찾습니다)
- 스키마부터 시작하는 경우 ex) `https://www.naver.com`

##### Relative URL 표현하기

- `/`로 시작하지 않는 경우 
- `../`로 시작하는 경우
